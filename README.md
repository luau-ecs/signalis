# signalis

signalis is a robust, fast, fully-featured signal library for luau. signalis is still early in development (pre-1.0),
expect breaking changes on updates to 0.**x**.0 versions.

## installation

you can install signalis via [wally](https://wally.run/package/luau-ecs/signalis?version=0.1.0). as of right now, signalis is only one file at `src/init.luau`

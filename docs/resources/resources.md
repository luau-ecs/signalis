# resources

## documentation

- [learn](../learn/contents.md) for general information and a walkthrough of the api
- [semantics](../semantics/contents.md) for gritty information about the mechanics of signalis, without too many implementation details.
- [internals](../internals/index.md) tries to describe the implementation details, data structures used, and the theories behind it all.

all documentation including [homework](#homework), [learn](../learn/contents.md), [semantics](../semantics/contents.md),
[internals](../internals/contents.md), etc. are open for contribution! more info can be found in
[contributing](../contributing/info.md)

## homework

visit [the repo](https://github.com/luau-ecs/signalis/blob/master/docs/homework) for guides and homework exercises. for
the full experience, first, clone the repo:
`git clone https://github.com/luau-ecs/signalis`

then, install with [`rokit`](https://github.com/rojo-rbx/rokit), navigate to `docs/homework` in your editor of choice,
and feel free to try out any of the exercises!
